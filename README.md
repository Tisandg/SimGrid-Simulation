# SimGrid Simulation

## Description
Simulation created using SimGrid v3.2

## Compilation
Check the path in the Makefile. Then execute the command `make`

## Execution
Run the following command
`ejemplo platform_cluster.xml 0.9`
`./ejemplo platform_cluster.xml 0.9`
